# ⚽ UCL Team Performance Analytics

A complete data analytics project that analyzes UEFA Champions League team performance using Python and Excel.

---

## 📌 Project Overview

This project focuses on transforming raw match data into a structured analytical dataset to evaluate team performance.

The workflow includes:

- Data cleaning and preprocessing
- Converting match-level data into team-level format
- Calculating performance metrics
- Generating a final analytical Excel report

---

## 🎯 Objectives

- Identify top-performing teams
- Analyze offensive and defensive strength
- Build a league-style ranking system
- Create a dataset ready for dashboards and reporting

---

- Pandas
